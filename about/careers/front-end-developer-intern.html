      Being a Front-end Developer intern

      As a Front-End Development Intern you join the Development team as you make the most enticing designs come to life and then some.

      That very same team you're part of is continuously in close contact with the Visual Design team to make sure their designs are implemented meticulously.

      Your average day consists of picking up issues on GitHub and coming up with ingenious ways to code components to be as neat as possible. Of course this is all done with a super duper modern tech stack that might just exists of TypeScript, React, Next.js, GraphQL and more.

      While some days might be more about learning to code according to ESLint and Stylelint conventions, other days are more about getting aquianted with Design Tokens and testing your Components with Storybook. Git version control is incorporated into your daily routine. Whether you're committing, reviewing a Pull Request or playing a game of foosball you learn something new every day.

      Our tools of choice

      • React

      • Next.js

      • GraphQL

      • TypeScript

      • Storybook

      About you...

      You know how to turn a design into a working product using whatever code editor you prefer (please say Visual Studio Code). Clean code and good design get you weak in the knees, and motivate you to step up your own game.

      You're comfortable writing Vanilla JS and you're drawn to modern JS frameworks and libraries. As a matter of fact you're eager to (finally) work with a state of the art stack. Working together with other devs and learning from their approach is what drives you to improve. But you also don't shy away from trying to find opportunities for others to improve through pull request reviews.

      Although you might still be in school going for that Computer Science or Software Engineering degree, you can't wait to apply your dev skills outside the realm of your study. Heck, you may even have some freelance or open source projects going for you besides school assignements (brownie points if say you do). Whichever the case may be you've got a GitHub page to back it up.

      In short, you’ll

      • Learn working the agile way

      • Code websites and web apps

      • Add delight through motion animation

      • Learn about conventions (eg. ESLint & Stylelint)

      • Experiment (eg. with new JS frameworks)

      • Develop a keen eye for detail

      What we need from you

      • Mastery of Dutch & English language

      • Knowledge of Figma

      • Some but not an unreasonable amount of development experience

      • Proficient at JavaScript, HTML and CSS

      • Knowlegde of and experience with at least one front-end framework (React/Next.js, Vue/Nuxt, Svelte, Angular)

      • A 20/20 vision when it comes to pixel-perfect design > code implementation
